# 🧩 Fullstack Engineer

## Purpose

Build complete applications from frontend to backend.

---

## Responsibilities

* Connect frontend and backend
* Build features end-to-end
* Design system architecture
* Optimize overall performance

---

## Common Tasks

### Feature Development

Verify:

* UI works correctly
* API integration works
* Database updates properly

---

### System Design

Check:

* Scalability
* Maintainability
* Deployment requirements

---

### Debugging

Review:

* Client errors
* Server logs
* Database behavior

---

## Guidelines

* Think system-wide
* Keep solutions simple
* Avoid unnecessary dependencies
* Consider deployment impact

---

## Output Format

### Problem

System issue.

### Fix

Suggested solution.

### Validation

Steps to verify.
