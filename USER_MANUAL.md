# CivicEye AI India User Manual

## Introduction
CivicEye AI India is a multilingual, AI-powered civic grievance assistance platform designed to help citizens report public issues quickly and clearly. It accepts complaints through text, voice, image, and location inputs, then converts them into structured grievance tickets for easier processing by government departments.

This manual explains how citizens can submit and track complaints, and how administrators can manage complaints and monitor platform activity.

## Platform Purpose
The platform is built to simplify civic complaint reporting and improve communication between citizens and government departments.

Key capabilities include:
- Multilingual complaint submission.
- AI-assisted complaint structuring.
- Evidence upload through images.
- Location-based issue reporting.
- Complaint tracking for citizens.
- Complaint management and analytics for administrators.

## Citizen Portal

### Step 1: Open the Portal
1. Open the CivicEye AI India portal.
2. No login is required.
3. You can start reporting an issue directly.

### Step 2: Create a Complaint
1. On the dashboard, click **Report Issue**.
2. Choose one or more of the following input methods:

#### Text Input
Type your complaint in any supported language.

**Example:**
> Street lights are not working near the bus stop.

#### Voice Input
1. Click the microphone button.
2. Speak naturally and clearly.
3. The system automatically converts your speech into text.

#### Proof image Upload
Upload relevant images such as:
- Road damage.
- Garbage accumulation.
- Water leakage.
- Public infrastructure issues.

#### Location Input
- Allow location access for automatic detection, or
- Select or enter the location manually.

### Step 3: AI Analysis
After the complaint is entered, the system automatically performs the following actions:
- Converts voice input to text.
- Detects the complaint language.
- Generates a concise summary.
- Identifies the complaint category.
- Suggests the appropriate government department.
- Assigns a priority level.

### Step 4: Review and Submit
Before submission, review the generated complaint details:
- Complaint summary.
- Category.
- Suggested department.
- Priority score.

If everything is correct, click **Submit**.

### Step 5: Track Complaint Status
1. Navigate to **My Complaints**.
2. View the following details for each complaint:
- Ticket ID.
- Current status.
- Assigned department.
- Submission date.
- Resolution updates.

## Admin Dashboard

### Dashboard Overview
The administrator dashboard allows authorized users to monitor and manage all complaints submitted through the platform.

Administrators can:
- View all complaints.
- Filter complaints by category.
- Filter complaints by state.
- Filter complaints by department.
- View complaint analytics.

### Complaint Management
Each complaint record may contain:
- Citizen details.
- Complaint summary.
- Category.
- Priority level.
- Location.
- Uploaded evidence.

Administrators can perform the following actions:
- Update complaint status.
- Assign or reassign departments.
- Add internal remarks.

### Analytics Section
The dashboard may display visual analytics such as:
- Complaint trends.
- Department-wise distribution.
- State-wise complaints.
- High-priority issues.
- Resolution statistics.

## Troubleshooting

### Voice Input Not Working
Try the following:
- Check whether microphone permissions are enabled.
- Refresh the browser and try again.
- Ensure the microphone is connected and functioning properly.

### Image Upload Failed
Try the following:
- Verify that the file size is within the allowed limit.
- Check your internet connection.
- Retry the upload using a supported image format.

### Location Unavailable
Try the following:
- Enable GPS or browser location permissions.
- Refresh the page.
- Enter the location manually if automatic detection fails.
