# CivicEye Citizen App

This is the citizen-facing interface for the CivicEye AI platform.

## Features
- Report issues via text, voice, and images.
- Automatic location detection.
- Multilingual support.
- Real-time tracking of complaint status.

## Tech Stack
- React
- Vite
- Axios
- Lucide React

## Getting Started
1. `npm install`
2. `npm run dev`
